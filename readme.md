# Dev Station - Browser Extension

> For developers by developers ❤️

### Develpment Usage
```sh
npm install 
npm run dev
```

### Considerations 
The funcionality behind the frontend is hosted at [dev-station-core](https://github.com/cr0wg4n/dev-station-core)
